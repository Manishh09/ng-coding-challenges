# ngc-signals

Angular Signals Challenges - covering Angular's reactive Signals API (signal, computed, effect), signal-based state management patterns, component communication with signals, performance optimization, and migration strategies from RxJS to Signals.

## Project Structure

```
src/app/
├── challenges/                    # All challenge implementations
│   ├── challenge-XX-[name]/       # Individual challenge folder
│   │   ├── components/            # Challenge workspace components
│   │   ├── services/              # Optional services
│   │   ├── models/                # Optional TypeScript interfaces
│   │   └── docs/                  # Challenge documentation
│   │       ├── CH-XX-REQUIREMENT.md
│   │       └── CH-XX-SOLUTION_GUIDE.md
├── app.component.ts               # Root component
├── app.config.ts                  # Application configuration
└── app.routes.ts                  # Routing configuration
```
